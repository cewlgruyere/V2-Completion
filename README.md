# V2-Completion
This BallsDex extension overrides the V3 completion command, and replaces it with one that has the new UI. Credits to the BallsDex team for the original code, the new code and also the utils used.

> [!NOTE]
> The original /balls completion command will not be available until you disable the package.

Installation:
```toml
[[ballsdex.packages]]
location = "git+https://github.com/cewlgruyere/V2-Completion.git@v0.0.1"
path = "v2_completion"
enabled = true
```
Paste this in the `config/extra.toml` folder. If you dont have it, create it and paste this in.

This package won't be maintained and was mostly made for fun!!! :3
